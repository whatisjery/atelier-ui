## API

| Name         | Type                              | Default     | Description |
| ------------ | --------------------------------- | ----------- | ----------- |
| children     | `ReactNode`                       | —           | Items to render in the column. |
| speed        | `number`                          | `1`         | Parallax intensity. |
| autoScroll   | `number`                          | `-0.2` | Auto-scroll speed. Negative values reverse direction. 0 to disable. |
| className    | `string`                          | `undefined` | Class for the container element. |
