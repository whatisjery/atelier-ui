## Usage

```tsx title="Imports"
import { InfiniteParallax } from "@/components/InfiniteParallax"
```

```tsx title="Usage example"
<div className="h-screen flex overflow-hidden">
    <InfiniteParallax>
        <img />
        <img />
        <img />
    </InfiniteParallax>
</div>
```

---

## API

| Name         | Type                              | Default     | Description |
| ------------ | --------------------------------- | ----------- | ----------- |
| children     | `ReactNode`                       | —           | Items to render in the column. |
| speed        | `number`                          | `1`         | Parallax intensity. |
| autoScroll   | `number`                          | `-0.2` | Auto-scroll speed. Negative values reverse direction. 0 to disable. |
| containerRef | `RefObject<HTMLElement \| null>`   | `undefined` | Scrollable container ref. Defaults is window. |
| className    | `string`                          | `undefined` | Class for the container element. |

---

## Notes

- Combine multiple columns with different `speed` and `autoScroll` values to create contrast.
- Pass `containerRef` when the component is not relying on the window scroll (like in the demo preview).
