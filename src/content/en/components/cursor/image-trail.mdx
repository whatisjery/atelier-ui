## Usage

```tsx
<ImageTrail>
  <img src="/1.jpg" alt="" className="w-32" />
  <img src="/2.jpg" alt="" className="w-32" />
  <img src="/3.jpg" alt="" className="w-32" />
</ImageTrail>
```

---

## API

| Name           | Type        | Default | Description    |
| -------------- | ----------- | ------- | ------------- |
| children       | `ReactNode` | —       | Items to cycle through as the cursor moves.|
| spawnDistance  | `number`    | `76`    | Distance in pixels before spawning an item.|
| driftAmount    | `number`    | `36`    | How far each item drifts from its spawn point in px.|
| removeDelay    | `number`    | `1.0`   | Seconds before a spawned item is removed.|
