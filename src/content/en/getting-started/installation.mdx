<DocStep title="Setup Tailwind CSS" step={1}>

This library uses Tailwind CSS v4. If your project doesn't have it, follow the [Tailwind CSS installation guide](https://tailwindcss.com/docs/installation) to get started.

</DocStep>

<DocStep title="License usage (optional)" step={2}>

Purchasing a license creates an account with access to the **source code**. Your **license key** lets you install **pro components** via the **CLI**.

Simply create or update your `.env` file in your project root and add your license key:

```text title=".env.local"
ATELIER_PRO_KEY=your-license-key
```
</DocStep>

<DocStep title="Installation" step={3}>

Find a component you like and choose whether to **copy and paste** the code directly into your project or **install it via the CLI**:

```text title="CLI example"
npx atelier-ui add fluid-distortion
```
</DocStep>
