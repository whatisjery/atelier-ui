## Set up Tailwind CSS

## Add the WebGL provider

If you are using WebGL components, install the provider and wrap your app root:

<InstalGuideCLI name="webgl-provider" />

```tsx title="Root layout"
import { WebglProvider } from "@/components/webgl-provider";

export default function RootLayout({ children }) {
  return <WebglProvider>{children}</WebglProvider>;
}
```

The provider runs one WebGL canvas for the whole page. Effects share it because separate WebGL contexts can't share resources or post-processing, so several effects duplicate GPU work and can't compose.

You still write each component in your markup like a normal element. The library reads where this element is and its size, then renders WebGL into that same space on the shared canvas, keeping it aligned as the page scrolls and resizes.

</DocStep>

<DocStep step={3}>

## Set up pro license

**Pro components** require a license. Copy your key from the panel at the top of this page and add it to `.env.local`.

```text title=".env.local"
ATELIER_PRO_KEY=<your-license-key>
```

Restart your dev server after adding the key.

</DocStep>

<DocStep step={4}>

## Install a component

Pick a component from the docs, then install it:

- **CLI** — installs the source and any shared dependencies.
- **Manual** — copy each file by hand from the component page.

```text title="CLI example"
npx atelier-ui add fluid-distortion
```

</DocStep>
