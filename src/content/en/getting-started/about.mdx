Atelier means **workshop** in French.

I built Atelier UI because I wanted a place to publish the effects and animations I work on (the kind you see on websites like Awwwards), and share them with other developers, instead of throwing them away. Components are built with React and are yours to copy, paste and use however you want.


The whole project is open source — feel free to [contribute](/docs/getting-started/contribution) if you want to get involved!
